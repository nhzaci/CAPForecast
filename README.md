# Module Forecast Calculator 
Pretty simple module forecast calculator to see projected grades to potentially maintain or drop your CAP

## Project setup
```bash
# Install dependencies 
$ npm install

# For hot reload
$ npm run serve

# Build server
$ npm run build

# Linting
$ npm run lint

# Deploy to github pages
$ npm run deploy
```
